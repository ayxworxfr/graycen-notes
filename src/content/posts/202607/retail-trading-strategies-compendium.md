## 前言：一个残酷的事实

95% 的散户在亏钱。

但剩下那 5% 并不是因为找到了什么"神奇公式"，而是因为他们做到了三件事：

```
① 有一套正期望的系统
② 严格执行，不临时起意
③ 活得够久，让概率发挥作用
```

这篇文章把市面上经过验证、散户可执行的交易策略做一个全景式整理。不是"教你一招制敌"，而是帮你**找到那个和你性格匹配的框架**。

## Part 1 / 趋势跟踪类

> *"截断亏损，让利润奔跑。"*

---

### 策略 1：双均线交叉系统

**最经典的趋势策略，没有之一。** 很多交易高手最终回归的终极武器，甚至简单到两根均线就够了。

```
规则极简：

短期均线（如 5日/10日）上穿长期均线（如 20日/60日）→ 买入
短期均线 下穿 长期均线 → 卖出

就这么简单。
```

| 参数组合 | 风格 | 特点 |
|:---:|:---:|:---|
| 5日 × 20日 | 短线 | 信号多，假信号多，适合活跃品种 |
| 10日 × 60日 | 中线 | 平衡型，多数人的起点 |
| 20日 × 120日 | 长线 | 信号少，一旦触发往往是大行情 |

**优化方向：**

- 加入成交量过滤（金叉时量能放大才有效）
- 加入趋势确认（ADX > 25 才执行信号）
- 震荡市暂停（横盘时均线缠绕，反复打脸）

**适合人群：** 有耐心、能忍受连续小亏等待大赚的人

---

### 策略 2：海龟交易法

上世纪 80 年代传奇实验中诞生的策略，至今仍有生命力。

```
核心规则：

买入 → 股价突破过去 20 日最高价
卖出 → 股价跌破过去 10 日最低价

加仓 → 每上涨 0.5 个 ATR，加一份仓位
止损 → 跌破入场价 2 个 ATR，无条件离场
```

**ATR（真实波幅均值）** 是这个策略的灵魂——它让仓位大小和止损距离根据市场波动自动调整：

```
波动大 → 仓位自动缩小 → 单次亏损可控
波动小 → 仓位自动放大 → 不浪费机会
```

| 优势 | 劣势 |
|:---|:---|
| 能抓到大趋势 | 胜率低（通常 35%-40%） |
| 风控极其严格 | 连续止损非常折磨心态 |
| 规则明确，无需主观判断 | 震荡市会反复被打脸 |

**核心启示：** 接受"错很多次"，只要"对的那次赚够多"就行。

---

### 策略 3：周线定方向 + 日线找入场

**大周期定方向，小周期找入场点。**

```
第一步：看周线

周线级别：
  → MACD 在零轴上方 = 多头市场，只做多
  → MACD 在零轴下方 = 空头市场，只观望

第二步：看日线

日线级别确认入场时机：
  → 回踩 20 日均线不破 + 缩量 → 买入
  → 放量跌破 20 日均线 → 止损
```

**这个策略的精髓在于"不做逆势交易"。** 周线空头时，哪怕日线看起来再好，也不出手。

## Part 2 / 均值回归类

> *"物极必反，否极泰来。"*

---

### 策略 4：布林带回归系统

利用布林带（BOLL）上下轨的统计学特性，在价格偏离均值过远时介入。

```
┌────────────────────────────────────────────┐
│                                            │
│  上轨 ─────── 超买区（考虑卖出）               │
│                                            │
│  中轨 ─────── 价值中枢（减仓/观望）            │
│                                            │
│  下轨 ─────── 超卖区（考虑买入）               │
│                                            │
│  核心前提：趋势未破坏（中轨向上或走平）          │
│                                            │
└────────────────────────────────────────────┘
```

**完整操作规则：**

```
买入条件（同时满足）：
  → 20 日均线走平或向上（趋势未坏）
  → 价格触及或跌破布林带下轨
  → MACD 绿柱缩短 / 出现底背离

卖出规则：
  → 反弹至中轨 → 止盈 50%
  → 反弹至上轨 → 清仓
  → 买入后再跌破下轨 3% → 止损离场
```

**关键细节：** 趋势向下时碰下轨不是"便宜"，是"还会更便宜"。必须加趋势过滤条件。

---

### 策略 5：RSI 极值反转

```
RSI(14) < 20  → 严重超卖，开始关注
RSI(14) < 20 后重新上穿 20 → 确认反转，买入

RSI(14) > 80  → 严重超买，准备离场
RSI(14) > 80 后跌破 80 → 确认见顶，卖出
```

**增强版：加入"RSI 背离"确认**

```
股价创新低，但 RSI 没有创新低 = 底背离 → 反转信号强
股价创新高，但 RSI 没有创新高 = 顶背离 → 见顶信号强
```

---

### 策略 6：恐慌性暴跌后的短线反弹

统计学上最强的均值回归信号之一：

```
触发条件：
→ 大盘或个股单日跌幅 > 7%
→ 成交量是平时 3 倍以上（恐慌性抛售）
→ 收盘出现长下影线（有资金承接）

操作规则：
→ 当日尾盘或次日开盘试探性买入（轻仓）
→ 目标：反弹 3%-5% 离场
→ 止损：再跌 3% 无条件走人
```

**注意：** 这是纯短线策略，赚的是情绪恢复的钱，绝不能变成"抄底变股东"。

## Part 3 / 动量轮动类

> *"强者恒强，弱者恒弱。"*

---

### 策略 7：相对强度轮动

**研究最充分、学术支撑最强的策略类型之一。**

```
逻辑：

过去 N 天涨幅排名靠前的品种，
未来继续上涨的概率大于随机水平。

操作（以 ETF 为例）：

① 每月末，计算所有候选 ETF 过去 20 天的涨幅
② 买入涨幅排名前 3 的 ETF
③ 等权持有一个月
④ 月末重新排名，轮换

空仓规则：
→ 如果排名前 3 的 ETF 涨幅都为负数
→ 全部持有货币基金（空仓等待）
```

| 参数 | 保守版 | 激进版 |
|:---:|:---:|:---:|
| 回看周期 | 60 天 | 20 天 |
| 持有周期 | 1 个月 | 2 周 |
| 持仓数量 | 3-5 只 | 1-2 只 |
| 换手频率 | 低 | 高 |

**核心是"追强不追涨"。** 不是在高位追单只股票，而是始终持有市场中最强势的那一批。

---

### 策略 8：RPS 排名选股法

```
RPS（相对价格强度）= 该股涨幅在全市场中的百分位排名

RPS > 90 → 跑赢了 90% 的股票 → 进入候选池
RPS < 50 → 弱于市场平均 → 坚决不碰
```

再叠加基本面过滤（CANSLIM 框架）：

```
RPS > 90
+ 营收增速 > 20%
+ 净利润增速 > 20%
+ 行业景气度向上
+ 机构持仓增加

= 高动量 + 高质量的交集
```

## Part 4 / 事件驱动类

> *"在确定性最高的时间窗口出手。"*

---

### 策略 9：可转债套利

A 股特有的制度红利——

```
打新策略（零成本正期望）：
→ 满足市值要求后参与申购
→ 中签后上市首日大概率有正收益
→ 无需分析，纯概率游戏

强赎套利：
→ 强赎公告发布后，转股价值已确定
→ 溢价率为负或极低时买入
→ 赚的是"确定性事件到来前的折价修复"
```

---

### 策略 10：业绩预增窗口

```
时间窗口：
→ 年报/季报披露前 1-2 周
→ 已有业绩预告且大幅预增（> 50%）

逻辑：
→ 业绩确定性高 + 市场尚未充分定价
→ 报告正式发布时可能有估值修复

关键风控：
→ "利好出尽是利空" 这种事经常发生
→ 必须在发布日前设好止盈/止损线
→ 绝不恋战
```

---

### 策略 11：定增破发策略

```
筛选条件：
→ 定增价格已知
→ 当前股价 < 定增价（定增参与者已浮亏）
→ 定增解禁日临近

底层逻辑：
→ 定增机构有动力在解禁前把股价拉回成本线
→ 本质是"搭别人的轿子"

止损纪律：
→ 跌破定增价 10% 以上 → 逻辑失效，离场
```

## Part 5 / 量化网格类

> *"不预测涨跌，只收割波动。"*

---

### 策略 12：等差网格

```
设定：
→ 确定价格区间（如 ETF 在 1.0 - 1.5 之间震荡）
→ 每隔 0.05 元设一个格子
→ 每跌一格，买入一份
→ 每涨一格，卖出一份

示意：

1.50 ┤ ←── 卖出
1.45 ┤ ←── 卖出
1.40 ┤ ←── 卖出
1.35 ┤      ★ 当前价格
1.30 ┤ ←── 买入
1.25 ┤ ←── 买入
1.20 ┤ ←── 买入
```

**进阶优化：**

```
等比网格：
→ 格距按百分比设定（如每跌 3% 买一份）
→ 越跌格子越密，平均成本更优

动态网格：
→ 根据 ATR 自动调整格距
→ 波动大时格子放宽，波动小时格子收窄
```

**天敌：** 单边下跌。必须选"跌不死"的品种（宽基 ETF），并设置整体止损线。

---

### 策略 13：条件单自动化执行

多数券商 App 支持条件单功能，可以实现半自动化交易：

```
预设规则：
→ 价格跌至 X 元 → 自动买入 N 股
→ 价格涨至 Y 元 → 自动卖出 N 股
→ 浮亏达 Z%  → 自动止损

核心价值：
→ 消除"临时改主意"的人性弱点
→ 开会、睡觉、上班时系统自动执行纪律
→ 把策略从"想法"变成"行为"
```

## Part 6 / 仓位管理与风控

> *"策略决定你能赚多少，风控决定你能活多久。"*

---

### 凯利公式（仓位上限参考）

```
f = (p × b - q) ÷ b

f = 最优仓位比例
p = 胜率
b = 盈亏比（平均盈利 ÷ 平均亏损）
q = 1 - p（败率）
```

**举例：**

```
胜率 40%，盈亏比 3:1

f = (0.4 × 3 - 0.6) ÷ 3 = 0.2

→ 每次最多投入总资金的 20%
```

**实战建议：** 凯利值再打半折使用（Half Kelly），因为真实参数永远有误差。

---

### 2% 法则（单笔最大亏损控制）

```
铁律：单笔交易最大亏损 ≤ 总资金的 2%

推导过程：
→ 总资金 10 万
→ 单笔最大亏损 = 2000 元
→ 止损距离 5%
→ 最大仓位 = 2000 ÷ 5% = 4 万

生存意义：
→ 连错 10 次，总资金损失 20%
→ 连错 20 次，总资金损失 40%
→ 你始终还有翻本的弹药
→ 而不是一把梭哈后再也站不起来
```

### 多策略组合（降低整体波动）

单一策略总有失效期。解决方案不是找"永远有效的策略"，而是**组合多个低相关策略**：

```
示例配置：

总资金 100%
├── 40% → 趋势策略（赚大行情的钱）
├── 30% → 均值回归策略（赚震荡市的钱）
├── 20% → 事件驱动策略（赚确定性事件的钱）
└── 10% → 现金（随时备用）

组合效果：
→ 趋势策略在震荡市亏钱时，均值回归在赚钱
→ 均值回归在趋势市踏空时，趋势策略在赚钱
→ 整体资金曲线更平滑
→ 心态更稳 → 执行力更强 → 正循环
```

## Part 7 / 策略选择指南

**没有最好的策略，只有最匹配的策略。**

---

### 速查表

| 你的特征 | 推荐方向 | 代表策略 |
|:---|:---|:---|
| 有耐心，能忍受长期空仓 | 趋势跟踪 | 海龟法、双均线 |
| 喜欢"捡便宜"的感觉 | 均值回归 | 布林带、RSI 极值 |
| 厌恶亏损，追求高胜率 | 网格 / 套利 | 等差网格、可转债 |
| 能接受低胜率高赔率 | 趋势突破 | 通道突破、动量 |
| 没时间盯盘 | 低频轮动 | 月度动量轮动 |
| 偏好确定性机会 | 事件驱动 | 定增破发、业绩窗口 |

---

### 选择前的五个自问

```
❶ 我能接受连续亏损几次？
   → ≤ 3 次 = 倾向高胜率策略（网格/回归）
   → ≥ 10 次 = 可以尝试趋势策略

❷ 每天能看盘多长时间？
   → < 30 分钟 = 周线级别 / 月度轮动
   → > 2 小时 = 日线级别策略

❸ 本金规模？
   → < 10 万 = 单策略集中执行
   → > 50 万 = 多策略组合降波动

❹ 能严格执行纪律吗？
   → 能 = 手动执行即可
   → 不能 = 条件单 / 程序化强制执行

❺ 目标年化收益？
   → 10%-15% = 稳健配置足够达成
   → 30%+   = 必须承受更大回撤
```

## 写在最后

写了这么多策略，最重要的一句话反而最简单：

> **策略本身只值 20 分。执行力值 40 分。风控值 40 分。**

太多人花 90% 的精力研究"什么时候买"，却只花 10% 的时间想"错了怎么办"。

真正在市场里存活下来的交易者，无一例外地把风控放在第一位：

```
先想最坏情况能不能承受
→ 能承受，再考虑怎么赚
→ 不能承受，换方案或不做
```

**活着，比什么都重要。**

**活得久了，复利会替你完成剩下的事。**
