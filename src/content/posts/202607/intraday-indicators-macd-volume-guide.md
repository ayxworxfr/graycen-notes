## 为什么需要看盘中指标？

很多人做交易，只看日线级别的 MACD、KDJ、布林带——这些指标解决的是"方向对不对"的问题。

但实际下单时，总会遇到一个尴尬处境：

```
"我知道该买，但到底是早盘买、还是尾盘买？"
"今天跌了 3%，是恐慌性下跌还是正常调整？"
"盘中拉了一波，能不能追？还是等回落？"
```

这些问题，日线级别的指标回答不了。

**分时指标，就是用来解决执行层面那"最后一公里"的。**

## 一、MACDFS：分时图上的 MACD

### 1.1 它是什么

MACDFS = MACD + FS（分时）。

算法和日线 MACD 完全相同（DIF、DEA、柱状图），只是把计算数据从日K线换成了当天的分时数据。

```
日线 MACD  → 反映过去几周到几个月的趋势动能
分时 MACDFS → 反映当天开盘以来的多空力量变化
```

### 1.2 三个组成部分

```
DIF（快线）：短期分时均线 - 长期分时均线
DEA（慢线）：DIF 的平滑移动平均
MACD 柱体 ：(DIF - DEA) × 2

红柱 → 多头占优，且力量在增强
绿柱 → 空头占优，且力量在增强
柱子由长变短 → 当前方向的力量正在衰竭
```

### 1.3 实战信号

**① 金叉与死叉**

```
DIF 上穿 DEA → 分时金叉 → 盘中短线买点
DIF 下穿 DEA → 分时死叉 → 盘中短线卖点
```

**② 零轴位置判断全天强弱**

```
MACDFS 始终在零轴上方运行 → 今天整体偏强
  → 回调不深，可以逢低接

MACDFS 始终在零轴下方运行 → 今天整体偏弱
  → 反弹乏力，逢高考虑减
```

**③ 分时背离（最有价值的信号）**

```
底背离：
  股价分时创出新低
  MACDFS 的 DIF 没有创新低
  → 空方力量衰竭，日内反弹大概率启动

顶背离：
  股价分时创出新高
  MACDFS 的 DIF 没有创新高
  → 多方力量衰竭，日内冲高大概率回落
```

分时背离是短线交易者日内择时最核心的工具之一。

## 二、分时均价线：一根线判断日内强弱

### 2.1 它是什么

分时均价线 = 当天累计成交金额 ÷ 当天累计成交量

它代表的是**当天所有参与者的平均持仓成本**。

### 2.2 怎么用

```
┌──────────────────────────────────────────────┐
│                                              │
│  股价在均价线上方运行                           │
│  → 今天买入的人平均都在赚钱                      │
│  → 抛压小，走势偏强                             │
│                                              │
│  股价在均价线下方运行                           │
│  → 今天买入的人平均都在亏钱                      │
│  → 套牢盘压力大，走势偏弱                        │
│                                              │
│  股价反复围绕均价线上下穿越                      │
│  → 多空胶着，方向未明，观望                      │
│                                              │
└──────────────────────────────────────────────┘
```

### 2.3 实战技巧

```
强势股特征：
  → 全天股价始终在均价线上方
  → 每次回踩均价线都被迅速拉起
  → 均价线本身稳步抬升

弱势股特征：
  → 全天股价压在均价线下方
  → 每次反弹触碰均价线就被打下来
  → 均价线本身逐步走低
```

## 三、量比：资金活跃度的温度计

### 3.1 计算方式

```
量比 = 当前每分钟成交量 ÷ 过去 5 日每分钟平均成交量
```

### 3.2 数值含义速查

| 量比范围 | 含义 | 操作参考 |
|:---:|:---|:---|
| < 0.5 | 极度缩量，市场无人关注 | 不参与，等待放量 |
| 0.5 - 1.0 | 温和缩量，维持惯性 | 正常观望 |
| 1.0 - 2.0 | 温和放量，正常活跃 | 可正常交易 |
| 2.0 - 5.0 | 明显放量，资金涌入 | 重点关注，可能有行情 |
| 5.0 - 10.0 | 剧烈放量，重大变化 | 谨慎对待，判断方向 |
| > 10.0 | 极端放量，通常对应涨停/跌停 | 不追，等消化后再决策 |

### 3.3 量比的组合用法

```
量比放大 + 股价上涨 → 真突破概率大
量比放大 + 股价下跌 → 恐慌性抛售或主力出货
量比缩小 + 股价上涨 → 无量空涨，持续性存疑
量比缩小 + 股价下跌 → 阴跌，暂无恐慌但也无承接
```

## 四、分时量柱：看资金在哪一刻发力

### 4.1 观察要点

分时图下方的量柱显示的是每一分钟（或每几分钟）的成交量。

```
突然出现大量柱 + 股价拉升 → 主力资金进场
突然出现大量柱 + 股价下跌 → 大单抛售
量柱均匀且矮小           → 散户之间的零碎交易
连续放量后突然缩量        → 一波行情可能结束
```

### 4.2 关键时段的量柱意义

```
09:30 - 09:45（开盘 15 分钟）：
  → 这段时间的量通常最大
  → 如果方向明确 + 量能充足 → 全天趋势大概率确立

13:00 - 13:15（午后开盘）：
  → 午后第一波量能方向往往决定下午基调

14:30 - 15:00（尾盘）：
  → 尾盘放量拉升 → 大资金抢筹，次日高开概率大
  → 尾盘放量下杀 → 机构出逃，次日低开概率大
  → 尾盘无量走平 → 无明显信号
```

## 五、委比与内外盘：多空意愿的微观观察

### 5.1 委比

```
委比 = (委买总手数 - 委卖总手数) ÷ (委买 + 委卖) × 100%

委比 > 0 → 挂单买盘多于卖盘 → 买方意愿更强
委比 < 0 → 挂单卖盘多于买盘 → 卖方意愿更强
```

### 5.2 内盘与外盘

```
外盘（主动买）：以卖一价成交 → 买方急于买入 → 看涨信号
内盘（主动卖）：以买一价成交 → 卖方急于卖出 → 看跌信号

外盘 >> 内盘 → 多方主动进攻
内盘 >> 外盘 → 空方主动出货
```

### 5.3 注意事项

```
⚠️ 委比和内外盘容易被操控：
  → 主力可以挂大单后撤单（虚假委比）
  → 主力可以对倒制造外盘假象

正确用法：
  → 只作为辅助参考，不单独决策
  → 结合价格走势和成交量一起看
  → 小盘股的委比参考价值低于大盘股
```

## 六、分时 KDJ：日内超买超卖

```
KDJ 参数通常为 (9, 3, 3)

K 值 > 80 + D 值 > 80 → 分时超买 → 短期冲高乏力
K 值 < 20 + D 值 < 20 → 分时超卖 → 短期杀跌动能衰竭

K 线上穿 D 线（低位金叉）→ 分时级别的买入时机
K 线下穿 D 线（高位死叉）→ 分时级别的卖出时机
```

## 七、各平台查看方式

| 平台 | 操作路径 |
|:---|:---|
| **同花顺** | 个股分时图 → 下方指标区域 → 点击切换为 MACDFS / KDJ |
| **通达信** | 个股分时图 → 右键 → 分时指标 → 选择 MACDFS |
| **东方财富** | 个股页 → 分时 → 底部指标栏 → 下拉选择 MACD分时 |
| **雪球** | 个股行情页 → 分时图 → 指标切换 |
| **手机券商 App** | 横屏分时图 → 底部指标栏 → 左右滑动切换指标 |

如果分时图指标栏没有显示 MACDFS，通常可以在"指标设置"中手动添加。

## 八、一套完整的日内看盘流程

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  【盘前】9:15 - 9:30                                         │
│                                                             │
│    → 观察集合竞价：量能大小 + 高开/低开/平开                      │
│    → 高开 2% 以上且量大 → 今天可能有行情                         │
│    → 大幅低开 → 观察是否有恐慌情绪可以利用                        │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【早盘】9:30 - 10:30                                        │
│                                                             │
│    → 看开盘 15 分钟量能和方向                                  │
│    → MACDFS 快速上零轴 → 偏强                                 │
│    → 量比 > 2 + 涨幅 > 2% → 资金明确流入                       │
│    → 第一波拉升后等待：看是否出现分时顶背离                       │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【盘中】10:30 - 14:30                                       │
│                                                             │
│    → 重点关注均价线的支撑/压力作用                               │
│    → MACDFS 零轴上下的切换 → 强弱转换信号                        │
│    → 缩量横盘 → 等待方向选择，不急于操作                          │
│    → 突然放量 → 判断是进攻还是出逃                               │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  【尾盘】14:30 - 15:00                                       │
│                                                             │
│    → 尾盘放量拉升 → 主力抢筹 → 可持仓过夜                        │
│    → 尾盘放量跳水 → 资金出逃 → 考虑减仓                          │
│    → 尾盘缩量走平 → 无增量信息 → 维持原计划                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 九、使用分时指标的三条纪律

```
纪律一：先看日线，再看分时

  日线/周线决定"做不做"
  分时只决定"什么时候做"
  绝不因为分时出现金叉就逆着日线方向操作

纪律二：不单独依赖任何一个分时指标

  MACDFS 金叉 + 均价线支撑 + 量能配合 = 高置信度
  只有 MACDFS 金叉，其他信号不配合 = 噪音

纪律三：承认分时指标的局限性

  分时数据噪音极大，一天内金叉死叉可能出现十几次
  主力可以通过对倒"画"出想要的分时形态
  小盘股的分时指标可靠性远低于大盘蓝筹和 ETF
```

## 总结

| 指标 | 解决什么问题 | 一句话用法 |
|:---|:---|:---|
| MACDFS | 盘中多空力量方向 | 零轴上方偏多，背离时反转 |
| 分时均价线 | 当天平均成本 | 价格在上方=强，在下方=弱 |
| 量比 | 今天的资金活跃程度 | >2 值得关注，<0.5 别碰 |
| 分时量柱 | 资金在哪一刻发力 | 放量看方向，缩量等信号 |
| 委比/内外盘 | 挂单意愿 | 仅作辅助，容易被操控 |
| 分时KDJ | 日内超买超卖 | 低位金叉买，高位死叉卖 |

---

> **最终原则：分时指标是"手术刀"，不是"指南针"。方向要靠日线和周线来定，分时只负责在对的方向上找到最精准的那一刀。**
