## 开场白

每个散户都经历过这样的剧本——

买入，跌了。割肉，涨了。追高，又跌了。

一次是运气差，两次是时机不对，但当这个剧本反复上演时，你不得不承认：**这不是巧合，这是设计。**

今天这篇文章，我想把"收割"这件事拆开来看，搞清楚镰刀的形状，然后聊聊我正在实践的一套用 ETF 做波段的防御性思路。

---

## Part 1 / 镰刀的五种形状

---

### 🔪 形状一：经典绞杀

这是最广为人知的模式，也是我最初以为的"唯一模式"。

完整剧本如下：

```
低位吸筹（安静地买，让你觉得这票没人要）
     ↓
拉升诱多（突然爆发，让你觉得机会来了）
     ↓
高位派发（你在买入，他在卖出，但股价还在涨）
     ↓
砸盘杀跌（出完货了，开始不计成本地砸）
     ↓
散户踩踏（你跑，他跑，所有人都在跑）
     ↓
低位回接（他用你割肉的价格，把筹码买了回来）
```

**一轮下来，筹码还是那些筹码，但钱换了口袋。**

---

### 🔪 形状二：时间差

你看到利好新闻兴奋地打开交易软件时，主力已经在三天前完成了布局。

你看到利空新闻慌张地挂出卖单时，主力已经在一周前跑得干干净净。

散户永远在"已知信息"上做决策，而主力在"未公开信息"上下注。

这不是交易能力的差距。**这是你根本不在同一个时间线上。**

---

### 🔪 形状三：画皮术

> *你看到的K线图，可能是画给你看的。*

MACD 金叉了？均线突破了？放量上涨了？

这些信号可以被资金"画"出来。主力知道散户看什么指标、信什么形态，于是：

- **做一个完美的突破形态 → 你追进去 → 砸**
- **做一个恐怖的破位形态 → 你割出来 → 拉**

技术分析不是无效的，但**当你的对手知道你在看什么时，它就变成了武器——对准你的武器。**

---

### 🔪 形状四：盘口魔术

买一挂着 5000 手大单，你觉得"有人护盘"，安心持有。

实际上那个大单是主力自己挂的幌子，他真正的出货单分散在上方十几个价位悄悄成交。

等你反应过来，大单撤了，股价已经跌穿你的成本线。

**你看到的"市场信息"，可能只是给你看的"舞台布景"。**

---

### 🔪 形状五：人性收割机

这把镰刀最高级，因为**它不需要操纵任何东西，只需要等你犯错。**

- 股价涨了三天，你终于忍不住追进去 → 你买在了情绪最亢奋的点
- 股价跌了两周，你终于受不了割出来 → 你卖在了情绪最绝望的点
- 板块轮动，你刚换过去的股第二天就开始跌 → 你永远慢半拍

不需要阴谋论。**散户的行为模式是统计学上可预测的，有人专门赚这个钱。**

---

### 小结

| 镰刀形状 | 核心武器 | 散户弱点 |
|:---:|:---:|:---:|
| 经典绞杀 | 筹码控盘 | 追涨杀跌 |
| 时间差 | 信息优势 | 后知后觉 |
| 画皮术 | 技术操纵 | 迷信指标 |
| 盘口魔术 | 资金优势 | 盲目跟单 |
| 人性收割机 | 等待 | 情绪决策 |

## Part 2 / 我的防御工事：ETF 波段思路

既然正面战场打不过，就不打正面战场。

我现在的做法是：**只做 ETF，只做波段，只在特定条件下出手。**

---

### 为什么选 ETF 而不是个股？

```
个股 = 丛林战（地形复杂，到处是埋伏）
ETF  = 开阔地（视野清晰，没有个股暗雷）
```

- ✅ 不存在"庄家控盘"——一篮子股票，谁也操纵不了
- ✅ 不存在"个股暴雷"——单只成分股出事，对整体影响有限
- ✅ 技术指标更"诚实"——没有人为画线的动机和能力

**换句话说：在 ETF 上，镰刀的形状二三四基本失效。**

---

### 策略核心逻辑

**一句话版本：**

> 在趋势没有走坏的前提下，价格触碰布林带下轨 + MACD 确认止跌时买入，反弹至中轨或上轨时分批卖出。

**视觉化：**

```
价格走势示意：

    ╱‾‾‾╲          ╱‾‾‾╲        ← BOLL 上轨（止盈区）
   ╱      ╲       ╱      ╲
──╱────────╲────╱────────╲───  ← BOLL 中轨（减仓区）
              ╲╱
               ⬆                 ← BOLL 下轨（买入区）
            这里买
```

---

### 选品原则（不推荐具体标的）

| 筛选维度 | 标准 |
|:---|:---|
| 流动性 | 日均成交额 > 1亿 |
| 长期逻辑 | 所跟踪的行业/资产有可理解的长期价值 |
| 波动率 | 有足够的波段空间覆盖交易成本 |
| 相关性 | 持有多只时，彼此相关性尽量低 |

**选什么品种是个人判断，我不做推荐。核心原则是：你能理解它为什么长期有价值，而不是因为"最近在涨"。**

---

### 买入条件（必须同时满足）

```
┌─────────────────────────────────────┐
│  条件 ①  趋势没走坏                    │
│  → 20日均线走平或向上                   │
│  → 股价在200日均线上方                  │
├─────────────────────────────────────┤
│  条件 ②  价格到位                      │
│  → 触及或跌破 BOLL 下轨                │
├─────────────────────────────────────┤
│  条件 ③  动能确认                      │
│  → MACD 绿柱连续缩短 / 底背离          │
│  → 或：收出止跌阳线 + MACD 金叉        │
└─────────────────────────────────────┘
         三个条件缺一不可
```

---

### 卖出规则

| 情境 | 动作 |
|:---|:---|
| 反弹至 BOLL 中轨 | 止盈 50% |
| 反弹至 BOLL 上轨 | 清仓 |
| MACD 红柱缩短 / 顶背离 | 清仓 |
| 买入后跌破下轨超 3% | **止损离场** |

---

### 仓位管理

不满仓。永远不满仓。

```
第一笔  ──────────  30%（条件触发，试探性买入）
第二笔  ──────────  30%（确认反弹，加仓）
预留    ──────────  40%（应对意外，保持灵活）
```

---

### 这套思路的天敌

我不回避它的局限性：

| 天敌 | 后果 | 应对方式 |
|:---|:---|:---|
| 单边暴跌 | 下轨不是底，越买越套 | 趋势过滤条件挡掉 |
| 单边暴涨 | 价格不碰下轨，全程踏空 | 接受，这不是策略目标 |
| 长期横盘 | 波幅太小，利润覆盖不了手续费 | 选波动率达标的品种 |

**任何策略都有不适用的场景。承认它，比假装它不存在重要得多。**

---

### 不同波动特征的参数思路

```
高波动品种（β > 1.2）
→ BOLL 参数：(20, 2.5)  放宽下轨，减少假信号
→ 操作级别：日线

低波动品种（β < 0.8）
→ BOLL 参数：(20, 2)  默认即可
→ 操作级别：周线更稳定
```

## Part 3 / 写在最后

回到开头的问题——主力为什么能收割散户？

不只是因为他们有钱，更因为**散户在一个被设计好的游戏规则里，用情绪做决策。**

我的应对不是"战胜主力"——这不现实。

我的应对是：

> **换一个他们够不到的战场（ETF），用一套不依赖情绪的规则（机械化条件），只在赔率对我有利时出手（下轨+确认），并且永远给自己留退路（止损+不满仓）。**

这不性感，不刺激，不会让你一夜暴富。

但它能让你**不再是那个被设计好的剧本里，准时出场的配角。**
